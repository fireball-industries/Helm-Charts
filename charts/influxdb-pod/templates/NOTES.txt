üöÄ QUICK START
================================================================================

1. Get your admin token (see above)

2. Test the API:
   curl -I {{ include "influxdb-pod.url" . }}/health

3. Write some data:
   influx write \
     --bucket sensors \
     --org {{ .Values.influxdb.organization }} \
     --token <YOUR_TOKEN> \
     --precision s \
     "temperature,sensor=sensor01,location=factory-floor value=23.5"

4. Query data:
   influx query \
     --org {{ .Values.influxdb.organization }} \
     --token <YOUR_TOKEN> \
     'from(bucket: "sensors") |> range(start: -1h)'
